1. Database: A database refers to the collection of datas that is electronically stored in a computer.

2. We have two types of database which are:
- Relational-database
- Non Relational-database

-- Relational-database: This refers to the collection and storing of data in a table which consist of rows and columns eg
S/N    |         Names        |          Email              |          Password
 1.         Precious Andrew       preciousand45@gmail.com           Precious345
 2.         Gabriel Jackson       Gabjack345@gmail.com              Jack456
 3.         Favour Benny          bennyfave879@gmail.com            Favour678
